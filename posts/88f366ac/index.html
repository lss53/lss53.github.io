目的

找到最适合自己的佳软。

前言

Windows平台始终是大多数人的第一生产力工具，是无可替代的一款产品。为了更加优雅地使用这款朝夕相伴的操作系统，特整理了一些优秀（知名）的软件，但愿你能找到最适合自己的。带标记为装机必备软件。

输入法

搜狗拼音

搜狗拼音是一款Windows平台下的汉字拼音输入法，为当前主流汉字拼音输入法之一，承诺永久免费。
@星空不寂寞 https://wwa.lanzoui.com/b04bhnwsb#8bts 密码:8bts

搜狗五笔

搜狗五笔是一款Windows平台下的汉字五笔输入法，为当前互联网新一代的五笔输入法，承诺永久免费。
@星空不寂寞 https://wwa.lanzoui.com/b04bhnwsb#8bts 密码:8bts

小狼毫

RIME/中州韵输入法引擎，是一个跨平台的输入法算法框架。基于这一框架，Rime开发者与其他开源社区的参与者在Windows、macOS、Linux、Android 等平台上创造了不同的输入法前端实现。Windows平台输入法前端名为小狼毫 (Weasel) https://bintray.com/rime/weasel/release 。
如果您是形码输入爱好者，建议使用 小狼毫修复工具 把输入法方案换成98五笔「需要双击字体98WB-2.otf」https://wwa.lanzoui.com/b04bhnyje#do4u 密码:do4u

视频音频

PotPlayer

来自韩国的视频播放器，总是最新版https://www.lanzoui.com/b0i38vsd

ShanaEncoder

免费强大的视频编码转换工具。简单使用说明https://www.52pojie.cn/thread-1107963-1-1.html。

打包完整绿色版步骤

  1. 去官网 https://shana.pe.kr/shanaencoder_download 下载最新版安装包ShanaEncoder.exe和编解码器库CodecLibrary.x64.7z「注：里包含了x86。本行略去了版本号，下同」。
  2. 用7-zip提取ShanaEncoder.exe到ShanaEncoder目录下，删除$PLUGINSDIR、ja-jp、zh-tw子目录。
  3. 进入presets目录，删除文件名里带韩文的子目录和文件名里带韩文的xml文件「其实预设一个都用不上，全删了都行」。
  4. 将CodecLibrary.x64.7z里的x64和x86俩目录提取到\ShanaEncoder\tools\里，对应覆盖到同名目录中。
  5. 打包完成。

文件下载

迅雷

支持多种协议的多线程下载工具，建议使用迅雷11。右键解压，本身不支持便携化的软件需手动运行绿化处理。

https://wwa.lanzoui.com/b04bhnyuf#hxpy 密码:hxpy

IDM

Internet Download Manager，全球最佳下载利器。

打包完整绿色版步骤

  1. 去官网 http://internetdownloadmanager.com/download.html 下载最新版安装包idman***build**.exe「注：*表示版本号」，双击安装。
  2. 去官网 https://www.internetdownloadmanager.com/support/toolbar2.html 下载PureFlat.zip、H3M_Glossy_Toolbar.zip文件。打开目录\Internet Download Manager\Toolbar删除里面所有文件，把PureFlat.zip、H3M_Glossy_Toolbar.zip解压到\Internet Download Manager\Toolbar。
  3. 进入%ProgramFiles(x86)%\Internet Download Manager\Languages只保留idm_chn2.lng，其它全部删除。
  4. 进入%ProgramFiles(x86)%\Internet Download Manager删除grabber.chm、idman.chm、idmcchandler2.dll、idmcchandler2_64.dll、IDMSetup2.log、license.txt、openssl-license.txt、scheduler.chm、tips.txt、tutor.chm文件。
  5. 进入%AppData%\Microsoft\Windows\Start Menu\Programs\Internet Download Manager和%HOMEDRIVE%\ProgramData\Microsoft\Windows\Start Menu\Programs\Internet Download Manager删除*Help.lnk、license.lnk、TUTORIALS.lnk文件。
  6. 去网盘 https://wwa.lanzoui.com/b04bhnzde#6sto 密码:6sto 下载绿化卸载.bat、IDMan v6.x Patcher (With no update).exe 「注：省略了版本号」，复制到 %ProgramFiles(x86)%\Internet Download Manager，双击IDMan v6.x Patcher (With no update).exe运行完毕后删除。
  7. 进入%ProgramFiles(x86)%\Internet Download Manager，Ctrl + A（全选），右键WinRAR => 添加到”Internet Download Manager”(T)，等待一下打包完成。
  8. 使用请解压后手动运行绿化处理。

微信

微信，超过10亿人使用。

  1. 去官网 https://dldir1.qq.com/weixin/Windows/WeChatSetup.exe 下载最新安装包WeChatSetup.exe。
  2. 去网盘 https://wwa.lanzoui.com/b04bho09g#cvs2 密码:cvs2 下载补丁微信多开补丁.exe、微信防撤回补丁.exe。

办公软件

Adobe Acrobat Pro DC

一款将世界上最优秀的PDF解决方案提升到新高度，其配有直观触控式界面及强大的新功能，可以将任何纸质文件转换为可编辑的电子文件，用于传输、签署和分享。

  1. 下载&安装程序：
    2017 https://helpx.adobe.com/acrobat/kb/acrobat-2017-downloads.html
    2020 https://helpx.adobe.com/download-install/kb/acrobat-2020-downloads.html
    Acrobat Pro 2017(or 2020) => For Windows => Download 。
  2. 安装时选择「使用“试用版”、“订阅或”指定用户许可证」即可。
  3. 激活程序：
    2017&2020 https://wwa.lanzoui.com/b04bhol3g#dmv4 密码:dmv4
    2017安装完毕后不要启动，把AMT Emulator v0.9.2.zip（amtemu.v0.9.2-painter.exe）拷贝到C:\Program Files (x86)\Adobe\Acrobat 2017\Acrobat目录下运行破解。
    2020安装完毕后不要启动，把Acrobat_Pro_DC_2019-2020_xxx_patch.rar（adobe.acrobat.pro.dc.2019.xxx-patch 13.11.2019.exe）拷贝到C:\Program Files (x86)\Adobe\Acrobat 2020\Acrobat目录下运行破解。
  4. 下载&安装补丁：
    2017 https://www.adobe.com/devnet-docs/acrobatetk/tools/ReleaseNotesDC/index.html#id3
    2020 https://www.adobe.com/devnet-docs/acrobatetk/tools/ReleaseNotesDC/index.html#classic-track
  5. 点击补丁网页中最上面的带update链接下载。
  6. 每次安装完补丁后，再次运行相应的激活程序。
